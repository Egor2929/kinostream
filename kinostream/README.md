# KinoStream

Бесплатный онлайн-кинотеатр: каталог фильмов и сериалов с обязательной преролл-рекламой перед каждым просмотром.

## Стек

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- [TMDB API](https://www.themoviedb.org/documentation/api) — каталог, поиск, метаданные

## Быстрый старт

1. Получите бесплатный API-ключ на [themoviedb.org](https://www.themoviedb.org/settings/api)

2. Создайте `.env.local`:

```env
TMDB_API_KEY=ваш_ключ
NEXT_PUBLIC_AD_DURATION=15
```

3. Запуск:

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Возможности

- Главная с трендами, новинками, популярным контентом
- Каталоги фильмов и сериалов с пагинацией
- Поиск по названию
- Страницы фильма/сериала с актёрами и похожим контентом
- **Обязательная преролл-реклама** (15 сек, без пропуска) перед каждым просмотром
- Плеер с автозапуском после рекламы

## Архитектура рекламы

`PreRollAd` — клиентский компонент с таймером и прогресс-баром. Готов к замене на VAST/AdSense:

- Подключите рекламную сеть в `src/components/watch/PreRollAd.tsx`
- Длительность настраивается через `NEXT_PUBLIC_AD_DURATION`

## Видеоисточники

Сейчас воспроизводятся официальные трейлеры с YouTube (через TMDB). Для полноценного стриминга подключите лицензированный CDN или собственное хранилище в `VideoPlayer`.
